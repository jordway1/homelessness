HUD Data

#Get HUD data from URL
url <- "https://www.hud.gov/sites/dfiles/Main/documents/2007-2019_PIT_Counts_by_CoC.XLSX"
response <- GET(url, write_disk(path <- tempfile(fileext = ".xlsx")))                               #!!Downloads temporary file to your disk!!

#Read each excel sheet into a list
homeless_list <- path %>%
  excel_sheets() %>%
  set_names() %>%
  purrr::map(read_excel, path = path)

#Standardize column names for each list element
for(i in 1:13){
  names(homeless_list[[i]]) <- str_remove(names(homeless_list[[i]]), ", \\d+")
}

homeless <- bind_rows(homeless_list[1:13], .id = "year")       #Bind rows of each list element to one object
names(homeless) <- str_replace_all(names(homeless), " ", "_")  #Replace spaces in variable names
homeless <- homeless %>% filter(CoC_Name != "Total")           #Filter out aggregate rows
rm(homeless_list, i, path, url, response)                      #Remove unnecessary objects

The homeless dataframe contains Point-In-Time estimates of homelessness from 2007 to 2019. Data is reported by Continuum of Care (CoC) providers. The CoC program is designed to promote communitywide commitment to the goal of ending homelessness. The program provides funding for and collects data from various nonprofit and government entities that help combat homelessness in the United States.

Each observation in the data represents local homelessness estimates by a CoC provider for a certain year. There are 408 unique CoC providers, and that has remained mostly constant since 2007. There are 5109 observations in the data with an average of 393 observations per year.

There are 80 columns in the data that describe various states of homelessness. Some examples include:

  • Overall_Homeless - the estimate of total homeless indivdiuals
  • Sheltered_Total_Homeless - the estimate of sheltered homeless individuals
  • Homeless_Veterans - the estimate of homeless veterans
  • Homeless_Family_Households - the estimate of homeless households

Analysis

The column CoC_Category is a categorical variable that can take the values: Rural CoCs, Suburband CoCs, Major Cities, or Other Urban CoCs. This data was made available in 2019, so all years prior have missing values for the variable. Since CoC providers have remained mostly constant throughout the years, I performed a self join to fill the column. Lets see how many CoC providers fall under each category and how many homeless individuals on average are accounted for by each type of provider.

#Self join to fill values of CoC_Category
homeless <- homeless %>% 
  filter(year == 2019) %>%

Homeless Demographics

The most basic question: how many people are homeless in the U.S. and how has that changed over time?

homeless %>%
  group_by(year) %>%
  summarize(count = sum(Overall_Homeless, na.rm = TRUE)) %>%
  ggplot(aes(year, count)) +
  geom_col(fill = "lightcyan4") +
  scale_y_continuous(labels = scales::comma) +

  theme_classic() +

As you can see, total homelessness has stayed around roughly 600,000 since 2007. That number began to decline around 2013, but has increased again in 2019 to 567,715. Lets take a look at how homeless estimates have changed over time, based on CoC_Category.

homeless %>% 
  group_by(year, CoC_Category) %>% 
  summarize(count = sum(Overall_Homeless, na.rm = TRUE)) %>%

Comparing Homelessness With Population and COVID-19 Data

In this section I will compare homelessness data to COVID-19 data which I downloaded from the New York Times COVID-19 github. I also obtained state population information from the U.S. Census Bureaus website. The following table shows the 10 states with the highest number of homeless people per 10,000, and includes COVID-19 information.

#Most recent case and death estimates for each U.S. State
covid_states <- read_csv("https://raw.githubusercontent.com/nytimes/covid-19-data/master/us-states.csv") %>%
  filter(date == "2020-09-07") %>%
  select(-c(fips, date))
#Population
state_populations <- read_csv("https://www2.census.gov/programs-surveys/popest/datasets/2010-2019/national/totals/nst-est2019-alldata.csv") %>%
  select(NAME, 'POPESTIMATE2019')

#Join these datasets to homeless and create columns for cases, deaths and homeless per 10,000 residents
homeless_states <- homeless %>%
  filter(year == 2019) %>%
  group_by(state) %>% 
  summarize(total_homeless = sum(Overall_Homeless, na.rm =TRUE)) %>%
  left_join(state_populations, by = c('state' = 'NAME')) %>%
  left_join(covid_states, by = 'state') %>%
  mutate(cases_per_10k = cases / (POPESTIMATE2019 / 10000), deaths_per_10k = deaths / (POPESTIMATE2019 / 10000), 
         homeless_per_10k = total_homeless / (POPESTIMATE2019 / 10000))
